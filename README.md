|   Variable   |               Usage                |
| :----------: | :--------------------------------: |
|   **UUID**   |             VLESS UUID             |
| **TR_PASS**  |          Trojan Password           |
| **PROXY_IP** | Proxy IP or domain (VLESS, Trojan) |
| **SUB_PATH** |         Subscriptions' URI         |
| **FALLBACK** |  Fallback domain (VLESS, Trojan)   |
| **DOH_URL**  |              Core DOH              |
